Formų tag'ai

  • form - forma
  • label - etiketė - naudojamas prie įvesties nurodyti įvesties laukelio pavadinimą
  • input - įvesties laukelis skirtas DAUG KAM (priklauso nuo type atributo reikšmės)
  • textarea - įvesties laukelis skirtas dideliam teksto kiekiui
  • select - įvesties laukas skirtas pasirinkti vienam iš pateiktų variantų
  • option - select'o pasirinkimo variantai
  • optgroup - option'ų grupės
  • output - formos išvesties vieta

Formos elementų atributai

  • for - label atributas - nurodo kuriam įvesties laukui priskirta label'is
  • id - * atributas - sujungti label su įvesties lauku; įvesties lauko atpažinimui per JS
  • name - įvesties lauko atributas - skirtas atpažinti įvesties lauką
  • placeholder - įvesties lauko atributas - skirtas atvaizduoti pavyzdinę įvestį
  • rows/cols - textarea atributas - skirtas nurodyti textarea dydį (geriau naudoti CSS'ą)
  • label - optgroup atributas - skirtas nurodyti option'ų grupės pavadinimui
  • value - * atributas - nurodo įvesties reikšmę
  • selected - option atributas - nurodo automatiškai parinktą option'ą
  • type - input atributas - daug opcijų:
    • button - mygtukas (nenaudojamas)
    • color - spalva
    • date - metų|mėnesio|dienos
    • datetime-local - metų|mėnesio|dienos|valanda
    • email - reikalauja email formato
    • file - pasirinkti kompiuterio failą
    • hidden - paslepia input'ą (nenaudojamas)
    • image - formos pateikimo mygtukas paveiksliuko pavidalu (nenaudojamas)
    • month - metų|mėnesio
    • number - skaičius
    • password - slaptažodis, kurį maskuoja
    • range - slankiklis
    • reset - išvalo formą
    • search - paieškai skirtas laukas
    • submit - mygtukas formos pateikimui
    • tel - telefono
    • text - tekstas
    • time - valandų pasirinkimas
    • url - nuoroda
    • week - savaitės pasirinkimas
    • checkbox - daugelis pasirinkimų
    • radio - vienas pasirinkimas
  • checked - radio|checkbox - nurodo automatiškai parinktą radio|checkbox

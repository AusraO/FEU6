Git ir GitHub

Git - kodo versijavimo programa

GitHub - svetainė, kurioje talpinamas versijuotas kodas ir juo dalinamasi tarp programuotojų

Nuoroda į GitHub
  1. Užsiregistruoti į GitHub'ą (susikurti paskyrą)
  2. Prisijungti prie GitHub'o

Komandinės eilutės komandos

  • TAB - užbaigia rašomą eilutę arba pasiūlo variantus (nenaudoti ant tuščios komandinės eilutės)
  • ls - išvardina direktorijas ir failus (list'ina)
  • cd - direktorijos keitimas (change directory)
  • CTRL C - atšaukia vykdomą komandą / stabdo procesus
  • clear - išvalo terminalo eilutes
  • direktorijosKūrimas, failųKūriams, trinimas, perkelimas/pervadinimas, failųKoregavimas
Nuoroda į Git

Git Komandos

  • init - sukuria git'o repozitoriją
  • add nurodytiFailai - pridėti koreguotus/naujus failus/aplankalus prie paruoštų failų kodo versijos kūrimui (commit'ui)
  • commit -m - iš paruoštų (staged) failų/aplankalų sukuriama kodo versija su atitinkama žinute (message (-m)), nusakančia kas toje kodo versijoje pakito

Funkcijos

Apsirašai funkciją: function f_Vardas(param1, param2){ veiksmai... }

Išsikvieti funkciją: f_Vardas(arg1, arg2);

Parametrai ir Argumentai

Parametrai yra duomenys, kuriuos funkcija priima ir naudoja savo viduje.

Argumentai yra duomenys, kurie yra perduodami funkcijai, jos kvietimo metu.

Callback

Kai funkcija ar metodas kreipiasi į kitą funkciją.

Arrow function (lambda)

Naujoviškesnis funkcijos aprašymo būdas išėjęs kartu su EcmaScript6.

Norint sukurti vardinę arrow funkciją, reikia - susikurti kintamąjį, kuriam prilygini bevardę arrow funkciją.

Bevardė arrow funkcija užsirašo: 1) (par1, par2, par3) => { veiksmai... } 2) parametras => { veiksmai... } 3) parametras => veiksmas;

Color names

Užvadintų spalvų yra ~140. Norint pritaikyti spalvą - reikia nurodyti jos pavadinimą. Pvz.: color: orange|black|khaki|dimGray...;

RBG(a) (red green blue)(alpha)

Naudojant RGB kodą, kiekviena spalva susideda iš 3 pagrindinių spalvų mišinio. Vienos spalvos kiekis nurodomas skaičiumi nuo 0 (nėra) iki 255(max). Pvz.: color: rgb(0, 255, 0)|rgb(50, 189, 255);

rgbA - red green blue ALPHA. Alpha nurodo spalvos permatomumą, priskiriant skaičių nuo 0 iki 1, 0.001 žingsniu. 0 - spalva pilnai permatoma (nematoma), 1 - spava pilnai matoma (tarsi nebūtų jokio alpha nustatymo). Pvz.: rgba(0, 255, 0, 0.501)|rgba(50, 189, 255, 0)

Hex kodas (#)

Naudojant Hex kodą, kiekviena spalva susideda iš 3 pagrindinių spalvų mišinio. Vienos spalvos kiekis nurodomas išraiška nuo 00 iki ff. Pvz.: color: #00ff00 | #32bdff;

Norint nurodyti spalvos permatomumą naudojant hex spalvos pavidalą, gale reikia nurodyti 2 papildomus simbolius (1 hexdecimal išraišką). 00 - visiškai permatoma, ff - pilnai matoma. Pvz.: color: #00ff0080 | #32bdff00;

Hex kodą galima trumpinti nurodant dvigubai mažiau simbolių. Trumpinimas galimas tik tuomet, jeigu yra du vienodi simboliai iš eilės - juos galima trumpinti į vieną simbolį. Pvz.: color: #0f0 (vietoj #00ff00) | #3bf (beveik vietoj #32bdff);

Kaip veikia Hex? 00 - 0; 01 - 1; 02 - 2; 09 - 9; 0a - 10; 0b - 11; 0c - 12; 0d - 13; 0e - 14; 0f - 15; 10 - 16; 11 - 17;... 19 - 25; 1a - 26;... 1f - 31; 20 - 32; 9f - ???; a0...... ff - 255;

Gradient

  • Linear
  • Radial
  • Conic
  • repeating...

Norint naudoti spalvų gradientus, reikia pasirinkti spalvos stilių ir jo nustatymuose nurodyti, kad naudojamas kažkuris gradientas. Pvz.: background-color: linear-gradient(nustatymai);

Nustatymai susideda minimum iš 3 dalių: kur prasideda gradiento piešimas | į kurią pusę piešimas vyksta; spalva1; spalva2;

Nurodant spalvas, galima nurodyti % reikšmę kiek vietos jie užims.

Positions

  • static
  • relative
  • absolute
  • fixed
  • sticky

Nustačius betkurią apart static poziciją, elementui galime suteikti: top, right, bottom, left ir z-index stilius.

Static

Static pozicija yra default'inė pozicija visų elementų. Dėl to ji nieko nedaro, net nesuteikia galimybės naudotis top, right, bottom, left ir z-index stiliais.

Relative

Relative pozicija suteikia elementui galimybę naudotis top, right, bottom, left ir z-index stiliais. Relatyviosios pozicijos top, right, bottom ir left nustatymai pastumia elementą per nurodytą atstumą, nuo nurodytos kraštinės, nuo elemento pradinės prozicijos. Elementas turintis relative poziciją yra jo viduje esančio elemento, su absoliučiąją pozicija, riba.

Absolute

Elementas turintis absoliučiąją poziciją - išeina iš puslapio tėkmės (netenka savo pozicijos elementų išdėstymo eilėje). Suteikia galimybę naudotis top, right, bottom, left ir z-index stiliais. Absoliučios pozicijos top, right, bottom ir left nustatymai, pastumia elementą nuo jo protėvinio elemento kraštų, jei šis turi relatyviąją poziciją.

Fixed

Elementas turintis fiksuotą poziciją - išeina iš puslapio tėkmės (netenka savo pozicijos elementų išdėstymo eilėje). Jis fiksuojasi nurodytoje pozicijoje ir scroll'inant puslapiu, juda kartu. Fiksuotos pozicijos top, right, bottom ir left nustatymai nurodo kur elementas bus prilipęs ekrane.

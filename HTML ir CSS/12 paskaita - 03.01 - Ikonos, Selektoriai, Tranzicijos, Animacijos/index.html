Ikonos

Ikonos savo tag'o neturi. Jos naudoja kitus tekstinius tag'us ir priskirdamos jiems klases, būna atvaizduojamos. Dažniausiai naudojami tag'ai yra: i (italic) ir span.
Norint naudoti ikonas, reikia prijungti vieną iš daugelio ikonų bibliotekų prie savo svetainės. Galimos bibliotekos: FontAwesome, Bootstrap, Google, MUI, Ant-Design, Tailwind, Bulma...

Norint stilizuoti ikonas, jas stilizuojame kaip tekstinius elementus. Spalva: color; Dydis: font-size; Fonas: background-color;

Selektoriai

  • Paprasti:
    • element - elemento - pvz.: p
    • class - klasės - pvz.: .className
    • id - id - pvz.: #ID_Name
    • everything - visko - pvz.: *
    • multiple - daugelio elementų - pvz.: div, .pavadinimas
    • inside - elementai elemento viduje - pvz.: div p
    • children - vaikiniai elemento elementai - pvz.: div > p
    • element with class/id - elementas su klase/id arba klasėmis/id - pvz.: div.hihi.haha#idVardas
    • next element - elementas einantis už elemento - pvz.: input + label
    • next elements - elementai einantys už elemento - pvz.: input ~ label
  • Atributai:
    • [attribute] - renkasi elementą turintį nurodytą atributą - pvz.: [class]
    • [attribute='reikšmė'] - renkasi elementą turintį nurodytą atributą su specifine reikšme - [class="bi"]
    • [attribute~='reikšmė'] - renkasi elementą turintį nurodytą atributą, kur viena iš reikšmių yra nurodyta reikšmė - pvz.: [class~='bi']
    • [attribute|='reikšmė'] - renkasi elementą turintį nurodytą atributą, kur už reikšmės padėtas brukšnelis ir extra tekstas - pvz.: [class|='bi']
    • [attribute^='reikšmė'] - renkasi elementą turintį nurodytą atributą, kur reikšmė prasideda nurodyta reikšme - pvz.: [href^='https://']
    • [attribute$='reikšmė'] - renkasi elementą turintį nurodytą atributą, kur reikšmė pasibaigia nurodyta reikšme - pvz.: [src$='.png']
    • [attribute*='simboliuSeka'] - renkasi elementą turintį nurodytą atributą, kur reikšmėje yra nurodyta simbolių seka - pvz.: [alt*='pav']
  • Pseudo Elementai:
    • parinktas::before - nurodo virtualų elementą prieš parinktą elementą
    • parinktas::after - nurodo virtualų elementą už parinkto elemento
    • parinktas::first-line - nurodo pirmąją eilutę parinkto elemento
    • parinktas::first-letter - nurodo iki pirmosios raidės parinkto elemento
    • parinktas::marker - parinktų sąrašo elementų žymės
    • parinktas::placeholder - parinktų formos įvesčių placeholder atributo tekstai
    • parinktas::selection - parinktame elemente žymimo turinio stiliai

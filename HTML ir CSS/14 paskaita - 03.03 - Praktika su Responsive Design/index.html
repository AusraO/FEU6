Matavimo vienetai

  • px - absoliutusis vienetas
  • rem - šakninis dydžio nustatymas (1rem === 16px)
  • em - dydžio nustatymas, kuris reaguoja į protėvinio elemento font-size dydžio nustatymą
  • vw - viewport width - svetainės ekrano plotis (1/100 dalis)
  • vh - viewport height - svetainės ekrano aukštis (1/100 dalis)
  • % - procentas - dydžio nustatymas priklausomai nuo tėvinio elemento
